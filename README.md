# JEnigma
[![Build Status](https://travis-ci.org/RaveNoX/jenigma.svg?branch=master)](https://travis-ci.org/RaveNoX/jenigma)

Utility to encrypt / decrypt JSON configs

## Usage

```bash
$ go get -u github.com/RaveNoX/jenigma
# For help
$ jenigma -h
```
